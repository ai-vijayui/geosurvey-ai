GeoSurvey AI sample files

This folder gives beginner users safe files to test the platform.

Files in this folder:
- sample-gnss-points.csv: GPS point list sample
- sample-total-station-points.csv: total station point list sample
- sample-drone-image-01.jpg: lightweight placeholder photo sample
- sample-drone-orthomosaic.tif: lightweight placeholder map image sample
- sample-lidar-cloud.las: placeholder 3D scan training file
- sample-lidar-cloud.laz: placeholder compressed 3D scan training file
- sample-site-boundary.dxf: placeholder land drawing sample
- sample-hybrid-readme.txt: explains how to test a mixed-data workflow

Important:
- Some heavy survey formats are placeholders so the app can teach the upload flow without a large repo download.
- You can still download them, upload them, and use them to learn the screen flow.
- Start with the CSV samples if you want the easiest first test.
